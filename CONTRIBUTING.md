# Contributing to Mako Mermaids Mod

Thanks for your interest in the Mako Mermaids Mod! Since this is a fan project, we appreciate any help to make it better.

## üõ† How Can I Contribute?

### 1. Reporting Bugs
Found a glitch? Let us know!
* **Check existing issues** first to see if it's already known.
* **Open an Issue** using the "Bug Report" template.
* **Be specific:** Tell us exactly what happened and which mod version you are using.

### 2. Ideas & Suggestions
Have a cool idea for a new power or item?
* Open an Issue using the **Feature Request** template.
* Describe your idea clearly. We love hearing community feedback, but please understand that we can't implement everything.

### 3. Textures & Models
We are always looking for better textures or 3D models!
* If you are good with pixel art or Blockbench, feel free to share your work.
* You can post them on our **Discord** or attach them to a GitHub Issue.
* **Important:** Please ensure you created the assets yourself.

### 4. Code Contributions (MCreator)
This mod is created using **MCreator**.
* Direct code Pull Requests (PRs) via GitHub are difficult to merge and might not work with our workspace.
* If you have a specific procedure or code fix you want to share, please contact us on **Discord** first to discuss how to implement it.

If you have any questions, just ask on our **Discord Server**.
