‚ùì Frequently Asked Questions (FAQ)

What Minecraft version is this for?

The mod is currently built for Minecraft 1.21.8 (NeoForge). We will update to newer versions as they are released.

Is there a version for Bedrock or a Plugin?

No. This is a Java Edition mod only.

Will there be a Fabric version?

No, there are no plans for a Fabric port at this time.

How do I download and install?

1. Get the .jar file from our Downloads Page.
2. Install NeoForge for 1.21.8.
3. Drag the file into your .minecraft/mods folder.

Why are characters like Lewis included?

To create the ultimate roleplay experience! Side characters help recreate the full story.

Can I send ideas or suggestions?

Yes! Share your feedback in the #feedback channel on our Discord Server.

Why do you use AI for development?

I am a solo developer and not a coding expert. I use AI as a tool to help me write the code and make sure everything works correctly.
