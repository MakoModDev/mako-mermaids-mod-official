# Security Policy

## Supported Versions

We focus our support on the latest version of the mod.

| Mod Version | Minecraft Version | Supported          |
| :---------- | :---------------- | :----------------- |
| **v2.1.x** | 1.21 (NeoForge)   | :white_check_mark: |
| Older       | Older             | :x:                |

## Reporting a Vulnerability

If you discover a security issue (e.g., an exploit that crashes servers or allows cheating), please **DO NOT** post it publicly.

**How to report:**
Please report the issue privately by opening a ticket or contacting the Admin on our [Discord Server](https://discord.com/invite/8N774c8Ujd).

**What happens next?**
* We will try to review your report as soon as possible.
* If confirmed, we will do our best to fix it in a future update.

Thank you for keeping the community safe!
